# Atividades a serem desenvolvidas durante estágio

As atividades a serem desenvolvidas durante o estágio serão, a princípio:

- Concepção e desenvolvimento de fluxo de dados (ETL / data pipeline) para processamento e publicação dos conjuntos de dados geridos pela Assessoria de Inteligência de Dados - AID/SPLOR.

- Sugestões de melhoria ao padrão de metadados ([Data Package](https://datapackage.org/overview/introduction/)) selecionado para documentação dos conjuntos de dados geridos pela Assessoria de Inteligência de Dados - AID/SPLOR.

- Auxílio na criação e execução de capacitações [Trilha-dev](https://trilhadev.planejamento.mg.gov.br/main/).
